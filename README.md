poreminion
==========

Additional tools for analyzing Oxford Nanopore minION data (built on top of poretools)



>Requirements
==========

poretools (https://github.com/arq5x/poretools)
HDF5 >= 1.8.7 (http://www.hdfgroup.org/HDF5/)
R >= 3.0.0
Python >= 2.7
rpy2 >= 2.4.2
h5py >= 2.0
pandas
matplotlib
